# qbit Consensus Specifications

Documents in this directory describe versioned qbit consensus behavior for
implementers and reviewers. They are intended to make the byte-level rules
auditable without treating a moving external document as normative.

Consensus is ultimately enforced by the reviewed qbit source. If an unreviewed
contradiction is found between a document here and the implementation named by
that document, do not silently change either side. Record the discrepancy and
subject any behavioral resolution to separate consensus review.
