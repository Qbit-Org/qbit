# Install or Build qbit

Release downloads, checksums, signatures, supported platforms, and public
verification instructions are announced through https://qbit.org for each
published qbit network or release.

Until a release-specific verification page and qbit signing key set are
published, this file is source-build guidance only. Do not use Bitcoin Core
download pages, release keys, or binary verification instructions for qbit.

For source builds, start with the platform-specific build guides under
[doc/build/](doc/build/):

- [Unix](doc/build/build-unix.md)
- [macOS](doc/build/build-osx.md)
- [Windows](doc/build/build-windows.md)
- [Windows with MSVC](doc/build/build-windows-msvc.md)
- [FreeBSD](doc/build/build-freebsd.md)
- [NetBSD](doc/build/build-netbsd.md)
- [OpenBSD](doc/build/build-openbsd.md)
